# Space invader proyect

Este proyecto es una iniciativa práctica y educativa diseñada para ¿el
entendimiento de las funciones fundamentales de JavaScript a través de su aplicación en el desarrollo de un juego básico.

El objetivo principal no es solo crear un producto funcional, sino utilizar la estructura del
juego como un laboratorio interactivo. Al simular mecánicas esenciales, observaremos cómo el código da vida al movimiento del jugador,
registra sus triunfos con la puntuación y reacciona instantáneamente ante cada obstáculo o interacción. De esta manera, cada pulsación de
tecla y cada cambio en la pantalla se convierte en una demostración en vivo de la lógica de JavaScript operando en un entorno dinámico y en tiempo real.
